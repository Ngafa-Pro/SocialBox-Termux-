# SocialBox-Termux- 
SocialBox-Termux  SocialBox é uma estrutura de ataque de força bruta [Facebook, Gmail, Instagram, Twitter]  
  
Instalação  
  
apt-get update   
apt-get install git 
git clone https://github.com/samsesh/SocialBox-Termux.git  
cd SocialBox-Termux  
chmod +x SocialBox.sh 
chmod +x install-sb.sh  
./install-sb.sh 
./SocialBox.sh
